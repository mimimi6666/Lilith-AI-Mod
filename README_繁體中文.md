# 莉莉絲 AI MOD 0.1.1 RC4

這是《The NOexistenceN of Lilith》桌寵的非官方社群 MOD。它加入 AI 對話、文字與語音輸入、中／日語音、補充無配音台詞、天氣與經過審核的本機電腦操作。

## RC4 更新

- 相容官方 Build `24273498` 的分類式設定介面。
- MOD 設定只會顯示在「控制」分頁，不再覆蓋其他設定。
- 修復 F6／F7 第二次無法觸發，以及按鍵重新綁定與 Esc 取消失效。
- 中文／日文內建與補充語音切換維持相互獨立。
- 加入已測試的千問聊天與語音辨識、供應商原生聯網搜尋，以及明確電腦指令的本機可靠路由。
- 應用啟動可從執行中程式、Windows 開始選單、Store／MSIX 註冊與捷徑解析，不寫死玩家路徑。
- 修復 API Key 視窗殘留滑鼠／鍵盤焦點，並將原生權益兌換與 API Key 輸入完全分離。
- 安裝器可選擇將隱私最小化的莉莉絲 Codex 橋接合併到既有 Hooks，不覆蓋其他 Hooks。

## AI 服務相容性

**目前仍建議優先使用 Gemini；它是 0.1.1-RC4 測試最完整、針對性優化最多的服務。** 角色人格、上下文、多語言輸出、日文顯示／語音分離、Google Search、AI 信件與自然語言電腦工具，皆主要以 Gemini API 開發。

千問已完成實際接入測試，預設以 `qwen3.7-plus` 聊天、`qwen3-asr-flash` 辨識語音，並支援供應商原生聯網搜尋及明確應用、媒體、截圖與受審核電腦指令的本機路由。可用性、免費額度與計費要求依玩家的阿里雲模型服務帳戶及地區而定。

OpenAI 與 DeepSeek 目前僅提供**實驗性的文字聊天相容層**。基本請求已實作，正常取得的文字回覆也可能由本機 GPT-SoVITS 朗讀，但尚未針對各模型、回應格式、額度、地區限制及後續 API 變更做完整測試。這兩個服務目前沒有接入供應商原生的聯網搜尋與工具呼叫；只有部分明確指令仍可由 MOD 在本機直接辨識。

`F6` 語音辨識在 Gemini 與千問模式會跟隨所選供應商；OpenAI 與 DeepSeek 目前仍使用 Gemini 辨識，因此語音輸入需要另外儲存 Gemini API Key。

## 一鍵安裝

1. 執行 `LilithAI-Mod-Setup.exe`。
2. 安裝器會從 Steam 登錄資訊與所有 Steam Library 自動找到遊戲；找不到時才需要按「瀏覽」選擇包含 `Lilith.exe` 的資料夾。
3. 保持需要的元件勾選，按「安裝／更新」。
4. 第一次安裝 AI 動態語音會下載獨立推理環境，可能需要數分鐘；不會使用或修改玩家電腦原有的 Python。
5. 啟動後在左下角莉莉絲圖示按右鍵，選擇「加入 API KEY」及模型供應商，再輸入玩家自己的密鑰。
6. 若勾選 Codex 橋接，Codex 下次啟動可能會要求審核並啟用三個莉莉絲 Hooks；安裝器只會合併莉莉絲項目，不會覆蓋其他 Hooks。

安裝包不含作者的 API Key、聊天紀錄、玩家姓名、日誌、截圖、訓練資料集或開發電腦路徑。更新不會覆蓋既有的 API Key、聊天記憶、按鍵與玩家設定。

## 操作

- 預設 `F7`：開啟文字輸入氣泡。
- 預設按住 `F6`：錄音；放開後辨識並送出。
- 兩個按鍵都能在遊戲設定中重新綁定。
- 遊戲語音可在中文／日語間切換，選擇會保留到下次啟動。
- 「進階電腦操作」預設關閉；玩家開啟後，AI 才能使用白名單內的開啟程式、媒體鍵、視窗、截圖、計時器、鎖定與睡眠等功能。

## 硬體與網路

- 核心 MOD：Windows 10/11 x64，無額外硬體需求。
- AI 對話與語音辨識：需要網路與玩家自己的 API Key。
- AI 動態語音：NVIDIA 顯示卡（建議 8 GB 以上 VRAM）速度較佳；沒有相容 NVIDIA 顯示卡時會安裝 CPU 版，建議至少 16 GB RAM，生成速度較慢。
- 完整補充台詞語音約 301 MB；動態語音模型包約 1.98 GB，安裝時另需空間建立推理環境。

## 隱私

- 對話文字與語音辨識錄音會送往玩家選擇的 AI 供應商，受該供應商條款與隱私政策約束。
- GPT-SoVITS 語音合成在本機 `127.0.0.1` 執行，不對外開放連接。
- 天氣自動定位使用公開 IP 的約略城市位置；MOD 不保存 IP 位址。
- 截圖只存到玩家的「圖片\Lilith Screenshots」，不會自動上傳給模型。
- MOD 不提供任意 PowerShell／CMD、刪除檔案、讀取密碼或剪貼簿內容等權限。

## 更新、修復與移除

- 再次執行相同或新版安裝器並按「安裝／更新」，即可修復缺件或升級。
- 「移除 MOD」只移除本 MOD 管理的檔案；API Key、聊天記憶與設定預設保留。
- 若要完全清除個人資料，移除後手動刪除：
  - `BepInEx\config\community.lilith.textinjector.cfg`
  - `BepInEx\data\LilithTextInjector\memory.json`
  - `BepInEx\data\LilithTextInjector\ai-note-state.json`

## 排錯

- MOD 日誌：`BepInEx\LogOutput.log`
- 語音主機日誌：`BepInEx\data\LilithTextInjector\voice-runtime\logs\voice-host.log`
- 語音安裝日誌：`BepInEx\data\LilithTextInjector\voice-runtime\voice-runtime-install.log`
- 若 Steam 更新遊戲後 MOD 無法載入，先重新執行安裝器；仍失敗時附上日誌回報。
