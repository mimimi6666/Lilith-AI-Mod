# リリス AI MOD 0.1.1 RC2

これは『The NOexistenceN of Lilith』のデスクトップマスコット向けに制作された、非公式のコミュニティ MOD です。

AI 会話、テキスト入力、音声入力、中国語／日本語音声、未収録台詞への追加音声、天気機能、および事前に確認されたローカル PC 操作機能を追加します。

## RC2 アップデート

* 公式 Build `24273498` のカテゴリ式設定画面に対応しました。
* MOD の設定項目は「コントロール」タブにのみ表示され、他の設定ページと重ならなくなりました。
* F6／F7 が一度しか反応しない問題、キー再割り当て、Esc によるキャンセルを修正しました。
* 中国語／日本語の内蔵音声と追加音声は、引き続き完全に分離して切り替えられます。

## AI プロバイダーの互換性

**現在は Gemini の使用を推奨します。0.1.1-RC2 で実際にテストされ、個別の最適化が行われているプロバイダーは Gemini のみです。** キャラクター性、会話コンテキスト、多言語出力、日本語の表示文と音声文の分離、Google Search、AI レター、自然言語による PC ツール操作は、Gemini API を中心に開発・検証されています。

OpenAI と DeepSeek は、現時点では**実験的なテキストチャット互換レイヤー**です。基本的なリクエストは実装済みで、正常に返された文章をローカル GPT-SoVITS が読み上げる場合もありますが、各モデル、応答形式、利用上限、地域制限、今後の API 変更を含む一連の動作は未検証です。これらのプロバイダー固有のウェブ検索やツール呼び出しにはまだ対応しておらず、一部の明確な命令のみ MOD 側でローカル処理されます。

`F6` の音声認識は現在、常に Gemini のマルチモーダル音声認識を使用します。チャットで OpenAI または DeepSeek を選択している場合でも、音声入力には別途保存された Gemini API キーが必要です。録音は文字起こしのため Gemini に送信され、OpenAI や DeepSeek には送信されません。

## ワンクリックインストール

1. `LilithAI-Mod-Setup.exe` を実行します。
2. インストーラーは、Steam のレジストリ情報およびすべての Steam ライブラリフォルダーからゲームを自動的に検索します。見つからない場合のみ、**参照**をクリックして `Lilith.exe` を含むフォルダーを選択してください。
3. 必要なコンポーネントにチェックが入っていることを確認し、**インストール／更新**をクリックします。
4. AI 動的音声を初めてインストールする際は、専用の推論環境がダウンロードされます。完了まで数分かかる場合があります。プレイヤーの PC に既にインストールされている Python を使用または変更することはありません。
5. ゲーム起動後、左下にあるリリスのアイコンを右クリックし、**API KEY を追加**とモデルプロバイダーを選択してから、プレイヤー自身の API キーを入力してください。

インストーラーには、作者の API キー、チャット履歴、プレイヤー名、ログ、スクリーンショット、学習用データセット、開発用 PC のパスは含まれていません。

アップデートによって、既存の API キー、チャットメモリ、キー設定、プレイヤー設定が上書きされることはありません。

## 操作方法

* デフォルトの `F7`：テキスト入力バブルを開きます。
* デフォルトの `F6` を長押し：音声を録音します。キーを離すと音声認識が行われ、その内容が送信されます。
* どちらのキーもゲーム設定から変更できます。
* ゲーム内音声は中国語／日本語を切り替えられます。選択した言語は次回起動時にも保持されます。
* **高度な PC 操作**はデフォルトで無効です。プレイヤーが有効にした場合に限り、AI は許可リスト内のアプリ起動、メディアキー操作、ウィンドウ操作、スクリーンショット、タイマー、PC のロック、スリープなどの機能を使用できます。

## ハードウェアおよびネットワーク要件

* コア MOD：Windows 10/11 x64。追加のハードウェア要件はありません。
* AI 会話および音声認識：インターネット接続と、プレイヤー自身の API キーが必要です。
* AI 動的音声：NVIDIA 製 GPU の使用を推奨します。VRAM は 8 GB 以上を推奨します。対応する NVIDIA GPU がない場合は CPU 版がインストールされます。最低でも 16 GB の RAM を推奨しますが、音声生成速度は遅くなります。
* 追加台詞音声の完全パッケージは約 301 MB です。
* 動的音声モデルパッケージは約 1.98 GB です。インストール時には、推論環境を作成するための追加容量も必要です。

## プライバシー

* 会話テキストおよび音声認識用の録音データは、プレイヤーが選択した AI プロバイダーへ送信され、当該プロバイダーの利用規約およびプライバシーポリシーが適用されます。
* GPT-SoVITS による音声合成は、ローカル環境の `127.0.0.1` 上で実行され、外部からの接続は受け付けません。
* 天気機能の自動位置取得には、公開 IP アドレスから推定されるおおよその都市情報が使用されます。MOD は IP アドレスを保存しません。
* スクリーンショットは、プレイヤーの `Pictures\Lilith Screenshots` フォルダーにのみ保存され、モデルへ自動送信されることはありません。
* 本 MOD は、任意の PowerShell／CMD コマンド、ファイル削除、パスワードの読み取り、クリップボード内容の読み取りなどの権限を提供しません。

## 更新、修復、削除

* 同じインストーラー、または新しいバージョンのインストーラーを再度実行し、**インストール／更新**を選択することで、不足ファイルの修復または MOD のアップデートができます。
* **MOD を削除**では、本 MOD が管理するファイルのみが削除されます。API キー、チャットメモリ、設定はデフォルトで保持されます。
* 個人データを完全に削除する場合は、MOD 削除後に以下のファイルを手動で削除してください。

  * `BepInEx\config\community.lilith.textinjector.cfg`
  * `BepInEx\data\LilithTextInjector\memory.json`
  * `BepInEx\data\LilithTextInjector\ai-note-state.json`

## トラブルシューティング

* MOD ログ：`BepInEx\LogOutput.log`
* 音声ホストログ：`BepInEx\data\LilithTextInjector\voice-runtime\logs\voice-host.log`
* 音声インストールログ：`BepInEx\data\LilithTextInjector\voice-runtime\voice-runtime-install.log`
* Steam によるゲーム更新後に MOD が読み込まれなくなった場合は、まずインストーラーを再実行してください。改善しない場合は、関連するログを添付して不具合を報告してください。
